# Flavor Notes & Ideas

Running collection of era flavor, jokes, names, and copy ideas. Anything
here is fair game for the flavor-review AI to enhance, cut, or riff on.
Mechanics live in DESIGN.md; this file is pure tone.

## Era flavor

### HTML Era (1998)
- Visitor counter in the footer that actually counts your session keystrokes.
- "Best viewed in Netscape Navigator 4.0 at 800x600" badge.
- Clients ask for: guestbooks, hit counters, MIDI autoplay, "under
  construction" GIFs, webrings.
- Tagline idea: "You hand-code it. There is no other way."

### PHP Era
- Everything is a forum. Every client wants a forum.
- Error flavor: white screen of death; "headers already sent" as a recurring
  joke in bug tickets.
- The database is named `db_final_v2_REAL`.
- Tagline idea: "Now with dynamic content. What could go wrong?"

### JavaScript Era
- Framework names for the deprecation mechanic (fake but evocative):
  Quandary.js, Mootlace, Angularish, Bricks.js, hoverboard.io.
- News-ticker deprecation headlines: "Quandary.js maintainer discovers
  surfing, archives repo."
- Tagline idea: "The page no longer reloads. Neither do you."

### Framework/SPA Era
- Build tool jokes: the loading bar for `npm install` is a mini idle event.
- Clients start asking for "an app" when they mean a website.
- Tagline idea: "It's just a website, but now it needs a build step."

### AI Era
- AI assistant product name candidates: Cognivore, ShipItGPT, Devin't,
  PairBear, Copilotski.
- The AI's commit messages are suspiciously confident: "fix everything".
- Its excuses when bugs surface: "That code was correct in my training data."
- Tagline idea: "You don't write code anymore. You apologize for it."
